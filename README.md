# Aplicativo de gerencia de finanças em ReactJs

## Ferramentas/Tecnologias Utilizadas

- Yarn
- ContextAPI
- Axios
- StyledComponent
- MirageJs
- TypeScript

## Como Utilizar

#### Clone este repositório

```bash
git clone https://github.com/cesarakt/money-app.git
```

#### Instale as dependências

```bash
yarn install
```

#### Execute a aplicação

```bash
yarn start
```
